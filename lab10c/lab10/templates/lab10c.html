Django

Installing the Framework

Installing the framework was very simple. All I had to do was run the command pip install Django==4.2.7 in the terminal to install the latest stable version of Django

Building This Page

To get started, I had to create a new Django project.

I first entered a directy where I want to store the application, then ran django-admin startproject lab10c in the terminal.

Then, I had to edit the lab10/urls.py file to link it to my views file. The lab10c/urls.py also had to be edited to include a path to the lab10/urls.py file.

The next step was to create an HTML file so I could actually edit my webpage properly. I did this by creating a new folder called templates and placing a lab10c.html file inside that folder.

I then had to edit the views.py file to render the HTML code. After that, I added the lab10 folder to the INSTALLED_APPS array in the settings.py file.

Finally, I ran py manage.py migrate and the server was up and running!

Difficulties Encountered

The only difficulties I encountered were the ones when I was trying to set up the project. After that, everything was a breeze. To combat this, I was just following a few different tutorials, trying to piece together what I actually needed. A specific problem I had was that my website was not being displayed, but one of the tutorials told me not to just use the IP given when the server starts, but include /project_name at the end of the URL.
